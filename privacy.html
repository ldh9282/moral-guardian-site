Privacy Policy

Last updated: April 2026

1. Overview

Moral Guardian ("the Extension") is a Chrome browser extension that analyzes text for moral, ethical, and legal considerations. This Privacy Policy describes how we handle user data.

2. Data We Process

The Extension processes the following data locally on your device:

  • Text you type — analyzed in real-time for moral scoring when the feature is enabled
  • Text you select — analyzed on-demand when you drag-select text on a webpage
  • Analysis history — up to 200 recent analyses stored locally in Chrome storage
  • Settings & preferences — language, theme, trigger mode, font size, and other UI preferences
  • API key — your Anthropic API key, stored locally and never transmitted to our servers

3. Data We Do NOT Collect

We do not collect, store, or transmit any personal data to our servers. Specifically:

  • We do not have a backend server that receives your data
  • We do not track your browsing history
  • We do not collect analytics or telemetry
  • We do not use cookies or tracking pixels
  • We do not sell or share any data with third parties
  • We do not store your API key on any server

4. Third-Party Services

The Extension communicates with the following third-party services using your own API key:

  • Anthropic API (api.anthropic.com) — to perform AI-powered moral analysis, logic analysis, fact-checking, and text improvement. Your text is sent directly from your browser to Anthropic's servers using your personal API key. Please review Anthropic's Privacy Policy.
  • Korea Supreme Court (portal.scourt.go.kr, glaw.scourt.go.kr) — to search for relevant court cases when legal violations are detected. Only legal terms are sent, not your personal text.

5. Data Storage

All data is stored locally using Chrome's chrome.storage.local API. This data stays on your device and syncs only if you have Chrome Sync enabled (controlled by your Chrome settings, not by us).

You can delete all stored data at any time through the Extension's settings panel:

  • Analysis history — delete individual items or clear all
  • Cost tracking — reset from the dashboard
  • All data — uninstalling the Extension removes all stored data

6. Permissions

The Extension requests the following Chrome permissions:

  • storage — to save your settings and analysis history locally
  • activeTab — to detect text input on the current page
  • tabs — to identify which website you're on for platform-specific analysis
  • alarms — to keep the background service worker active during analysis
  • host_permissions (api.anthropic.com) — to call the Anthropic API for AI analysis

7. Children's Privacy

The Extension is not directed to children under the age of 13. We do not knowingly collect data from children.

8. Changes to This Policy

We may update this Privacy Policy from time to time. The "Last updated" date at the top reflects the most recent revision. Continued use of the Extension after changes constitutes acceptance of the updated policy.
